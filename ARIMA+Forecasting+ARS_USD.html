Comprobando que la serie temporal es estacionaria.¶

Por qu es importante?

En una serie de tiempo estacionaria, las propiedades estadsticas deben ser constantes en el tiempo y la autocovarianza debe ser dependiente del tiempo. Normalmente, cuando se corre una regresin normal, uno espera que las observaciones sean independientes entre s. Sin embargo, en una serie temporal uno tiene certeza que las observaciones dependen del tiempo. Para poder usar tcnicas de regresin en variables dependientes del tiempo la data debe ser estacionaria. Las tcnicas que aplican a variables independientes aleatorias tambin aplican a variables aleatorias estacionarias.

Hay dos formas de comprobar si una serie de tiempo es estacionaria. La primera es plotear la varianza mvil y observar si se mantiene constante en el tiempo; aunque siempre es posible hacer inferencias visuales. La otra forma es usar el test de Dickey-Fuller, un test estadstico cuya hiptesis nula es que la serie temporal es no-estacionaria, si el test da como resultado que la estadstica del test es significativamente menor que el valor crtico preestablecido, podemos rechazar la hiptesis nula en favor de tomar la serie temporal como estacionaria.

A continuacin, se calcula la varianza mvil, se plotean los resultados y se aplica el test de Dickey-Fuller en la serie temporal:

In [5]:
def test_stationarity(timeseries):
    
    #Estadsticos mviles
    rolmean = timeseries.rolling(window=52,center=False).mean() 
    rolstd = timeseries.rolling(window=52,center=False).std()

    #Plot:
    orig = plt.plot(timeseries.index.to_pydatetime(), timeseries.values, color='blue',label='Original')
    mean = plt.plot(rolmean.index.to_pydatetime(), rolmean.values, color='red', label='Media mvil')
    std = plt.plot(rolstd.index.to_pydatetime(), rolstd.values, color='black', label = 'Desvo mvil')
    plt.legend(loc='best')
    plt.title('Media y desvo estandard mviles')
    plt.show(block=False)
    
    #Test de Dickey-Fuller :
    print ('Resultados de Test de Dickey-Fuller:')
    dftest = adfuller(timeseries, autolag='AIC')
    dfoutput = pd.Series(dftest[0:4], index=['Estadstica del test','P-valor','#Lags','Nmero de observaciones'])
    for key,value in dftest[4].items():
        dfoutput['Valor Crtico (%s)'%key] = value
    print (dfoutput)

Estacionarizando la serie temporal¶

Si la serie de tiempo revela una tendencia o estacionalidad (no confundir estacional con estacionario), es una indicacin de que es no-estacionaria. Para estacionarizar la serie temporal se calculan la tendencia y estacionalidad y se remueven estos factores del modelo. Se pueden aplicar diferentes transformaciones para estacionarizar la data como se ver a continuacin.

Encontrando parámetros óptimos y construyendo el modelo ARIMA¶

Para aplicar ARIMA a la serie temporal es necesario encontrar los valores ptimos para los tres parmetros siguientes (p,d,q):

  • (p) El nmero de trminos autoregresivos (AR): Los trminos AR son simplemente desfases de la variable dependiente (el tipo de cambio en este dataset). Entonces, p=2 significa que los predictores de x(t) son x(t-1) y x(t-2).

  • (q) El nmero de trminos de media mvil (MA) : Los trminos MA son errores desfasados de prediccin en la ecuacin predictiva. Por ejemplo, con q=2 los predictores de x(t) son e(t-1) y e(t-2) donde e(i) es la diferencia entre la media mvil en el instante isimo y el valor verdadero.

  • El nmero de diferencias (d): Estos son los nmeros de diferencias no estacionales. En nuestro caso, d=1ya que estamos usando una serie de tiempo diferenciada en el primer orden.

Hay dos maneras de determinar el nmero de trminos AR y MA. La primera es usar la funcin arma_order_select_ic de Python. La segunda utiliza los grficos de la funcin de autocorrelacin (ACF) y la funcin de autocorrelacion parcial (PACF).

En este grfico, los valores 'p' y 'q' pueden ser determinados como sigue:

  • p: El valor donde la curva PACF corta el 0 por primera vez. (p=2)

  • q: El valor donde la curva ACF cruza el intervalo de confianza superior por primera vez. (q=2, q=1 tambien podra funcionar)

Esto significa que los valores ptimos para el modelo ARIMA(p,d,q) son (2,1,2).

Si los valores sugeridos por los grficos de ACF y PACF difieren de los valores obtenidos por la funcin arma_order_select_ic, se deben probar valores para los trminos p y q y usar los resultados del modelo ajustados para obtener sus valores AIC y proceder con con el modelo que tenga un menor valor de AIC.

Haciendo predicciones en la serie temporal¶

Lo que se ha logrado hasta este punto es una prediccin dentro de la muestra usando ARIMA, al entrenar el modelo en la toda la data de la serie temporal. Lo siguiente es separar el dataset en dos conjuntos, uno de training y otro de test. Se usar el dataset de training para entrenar el modelo ARIMA y realizar una prediccin fuera de la muestra. Luego se compararn los resultados de las predicciones fuera de la muestra con los valores del conjunto de test.

Se usar la funcin "forecast" para realizar una prediccin mvil de un paso con ARIMA. Una prediccin mvil es requerida dada la dependencia en las observaciones durante la diferenciacin y el modelo AR. Se re-crear el modelo ARIMA despus de que cada nueva prediccin sea recibida. Manualmente se mantendr registro de todas las observaciones en una lista llamada history, la cual tiene una semilla en la data de training y para la cual nuevas predicciones son incorporadas en cada iteracin.

In [19]:
size = int(len(ts_week_log) - 25)
train, test = ts_week_log[0:size], ts_week_log[size:len(ts_week_log)]
history = [x for x in train]
predictions = list()

print('Valores predichos vs Valores Esperados')
print('\n')
for t in range(len(test)):
    model = ARIMA(history, order=(2,1,1))
    model_fit = model.fit(disp=0)
    output = model_fit.forecast()
    yhat = output[0]
    predictions.append(float(yhat))
    obs = test[t]
    history.append(obs)
    print('Prediccin=%f, Esperado=%f' % (np.exp(yhat), np.exp(obs)))

error = mean_squared_error(test, predictions)

print('\n')
print('Error medio cuadrtico de las predicciones')
print('MSE: %.6f' % error)

predictions_series = pd.Series(predictions, index = test.index)
